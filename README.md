## 向量数据库
langchain教程里面用的的是Chroma向量数据库

### Chroma数据库使用
指定chroma持久化的目录，当我们不知道目录时,chroma会将数据存储在内存中，随着程序的关闭就会删除
`persist_directory = "./public"`
